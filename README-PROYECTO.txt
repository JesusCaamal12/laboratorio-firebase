# 🧪 MonitorLab - Sistema de Monitoreo de Laboratorios con Sensores

MonitorLab es una aplicación web desarrollada con **React** que permite visualizar y monitorear en tiempo real las condiciones de distintos laboratorios mediante sensores integrados. Ideal para instituciones educativas, centros de investigación o cualquier espacio que requiera supervisión constante de variables ambientales.

## 🚀 Características

- Visualización en tiempo real de datos de sensores (temperatura, humedad, CO₂, etc.)
- Panel de control por laboratorio con gráficos dinámicos
- Alertas cuando los valores superan los umbrales definidos
- Integración con APIs o servicios MQTT/WebSocket
- Interfaz moderna y responsiva

## 🛠️ Tecnologías utilizadas

- **React** con Vite
- **Chart.js** para visualización de datos
- **Tailwind CSS** para estilos rápidos y personalizables
- **Axios** para consumo de APIs REST
- Opcional: MQTT.js o Socket.IO para comunicación en tiempo real
